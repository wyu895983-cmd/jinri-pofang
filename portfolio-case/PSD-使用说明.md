# 今日破防作品集 PSD 使用说明

## 编辑文本

1. 在 Photoshop 图层面板展开对应模块组。
2. 找到名称以“标题｜”“文案｜”“功能说明”等开头的文字图层。
3. 使用文字工具直接修改内容、字体、字号和颜色。
4. 首次打开时 Photoshop 可能提示更新文字图层，确认更新即可。

## 替换截图

1. 展开对应模块组，找到名称以“截图｜”开头的图片图层。
2. 将新截图拖入 PSD。
3. 按住 `Shift` 等比例缩放到原图层大小和位置。
4. 隐藏或删除旧截图图层，并将新图层改为相同名称。

所有截图均来自 `portfolio-screens/`，在 PSD 中保持独立图片图层。

## 编辑模块

- 默认显示“背景｜完整视觉预览（隐藏后编辑模块）”图层，保证打开文件即可看到完整案例板。
- 需要逐层编辑时，隐藏该背景预览层，再显示并展开对应模块组。
- 每个模块组可独立移动、隐藏、复制或重排。

## 导出 PNG 或 JPEG

1. 在 Photoshop 中选择 `文件 > 导出 > 导出为`。
2. 格式选择 `PNG` 或 `JPG`。
3. PNG 适合保留最高画质；JPEG 建议品质设置为 `90-100`。
4. 确认画布尺寸为 `1920 × 7951 px` 后导出。

## 重新生成 PSD

```powershell
node scripts/generate-portfolio-psd.mjs
```
