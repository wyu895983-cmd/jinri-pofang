# 今日破防品牌图标体系

## Core Concept

一句话让人破防。

主图形采用“破裂聊天框”：聊天气泡代表吐槽社区，中心裂纹代表情绪被一句话击穿，右上角小粒子代表情绪释放和共鸣扩散。

## Design Rules

- 不使用文字“破”
- 不使用复杂插画
- 不使用多个表情
- 48x48 下仍保持清晰
- 深色底适配 iOS / Android 主屏幕
- 图形可延展到等级、勋章、动效和周边

## Extension Logic

- Lv1: 一道轻微裂纹
- Lv10: 轻微破裂，出现主辅双裂痕
- Lv30: 严重破裂，裂纹向外扩展
- Lv50: 爆裂，情绪释放线增多
- Lv100: 情绪核爆，气泡能量外扩
